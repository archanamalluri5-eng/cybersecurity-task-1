# Screenshots

> **ApexPlanet Cybersecurity Internship — Task 1**

This directory contains screenshots and visual evidence of lab activities.

## How to Add Screenshots

1. Take screenshots during your lab exercises
2. Save them with descriptive names (e.g., `kali-ip-config.png`, `nmap-scan-results.png`)
3. Place them in this directory
4. Reference them in your reports using relative paths

## Naming Convention

Use lowercase with hyphens: `category-description.png`

Examples:
- `lab-kali-desktop.png`
- `nmap-tcp-syn-scan.png`
- `wireshark-http-capture.png`
- `vulnerability-high-finding.png`
