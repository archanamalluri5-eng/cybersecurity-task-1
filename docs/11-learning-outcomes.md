# 11 — Learning Outcomes

> **ApexPlanet Cybersecurity Internship — Task 1: Foundation & Environment Setup**
> Summary of knowledge and skills gained across Days 1–12.

---

## Skills Acquired

### 1. Cybersecurity Fundamentals
- Understood the CIA Triad (Confidentiality, Integrity, Availability)
- Identified common threats: phishing, malware, DDoS, SQL injection, brute force, ransomware
- Recognized attack vectors: social engineering, wireless attacks, insider threats

### 2. Lab Environment Setup
- Installed and configured VirtualBox/VMware
- Deployed Kali Linux and Metasploitable2 VMs
- Configured isolated Host-Only network
- Verified connectivity and network isolation

### 3. Linux Proficiency
- Navigated the Linux file system (pwd, ls, cd, find)
- Managed files and directories (mkdir, cp, mv, rm)
- Understood and modified file permissions (chmod, chown)
- Used package management (apt, dpkg)
- Executed networking commands (ip, ping, ss, traceroute)

### 4. Networking Knowledge
- Studied the OSI Model (7 layers) and TCP/IP Model (4 layers)
- Compared TCP vs UDP protocols
- Understood DNS resolution and record types
- Learned IP addressing, private ranges, and subnetting basics
- Traced the TCP three-way handshake

### 5. Cryptography Awareness
- Understood symmetric (AES) and asymmetric (RSA) encryption
- Learned hashing concepts (MD5 vs SHA-256)
- Studied digital certificates and SSL/TLS
- Demonstrated encryption/decryption with OpenSSL

### 6. Security Tool Proficiency
- **Wireshark:** Captured and analyzed HTTP, FTP, DNS, and TCP traffic
- **Nmap:** Performed host discovery, port scanning, service detection, and OS detection
- **Burp Suite:** Familiarized with HTTP proxy and web application testing concepts
- **Netcat:** Used for low-level network communication and banner grabbing

### 7. Reconnaissance Skills
- Differentiated passive and active reconnaissance
- Performed WHOIS, DNS lookups, and Google OSINT awareness
- Understood Shodan and internet asset discovery

### 8. Scanning & Assessment
- Performed TCP SYN, UDP, service detection, and OS detection scans
- Documented scan results professionally
- Identified vulnerabilities with severity ratings
- Recommended remediation steps

### 9. Traffic Analysis
- Analyzed HTTP, FTP, DNS traffic in Wireshark
- Observed TCP three-way handshake
- Applied display filters for targeted analysis
- Identified security risks of unencrypted protocols

### 10. Ethical Awareness
- Understood legal and ethical boundaries of cybersecurity testing
- Committed to authorized-only testing practices
- Documented all activities professionally
