# Scripts

> **ApexPlanet Cybersecurity Internship — Task 1**
> Safe educational demonstration scripts for authorized local lab use only.

## Overview

This directory contains shell scripts that demonstrate cybersecurity concepts in a safe, controlled environment. All scripts operate exclusively on local test data and do not scan, attack, or interact with any external or unauthorized systems.

## Scripts

| Script | Description |
|--------|-------------|
| `linux-basics-demo.sh` | Demonstrates basic Linux commands and system information |
| `network-enumeration-demo.sh` | Shows local network interface information and lab connectivity |
| `crypto-demo.sh` | Demonstrates encryption, decryption, and hashing using OpenSSL |

## Usage

```bash
# Make scripts executable
chmod +x *.sh

# Run any script
./linux-basics-demo.sh
./network-enumeration-demo.sh
./crypto-demo.sh
```

## Safety Notice

- All scripts use only dummy/local data
- No public networks or systems are contacted
- No real credentials or sensitive data are used
- Scripts are designed for educational demonstration purposes only
